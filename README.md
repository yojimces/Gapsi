# Gapsi e-Commerce - Proyecto FullStack

Proyecto de evaluación técnica para Gapsi e-Commerce desarrollado con React 19 y Spring Boot.

## Estructura del Proyecto

Este repositorio contiene:

- **gapsi-front**: Aplicación frontend desarrollada con React 19, TypeScript, Vite, Redux Toolkit y Material-UI v7
- **gapsi-backend**: API REST desarrollada con Spring Boot (ver repositorio separado si aplica)

## Tecnologías Utilizadas

### Frontend
- React 19 con TypeScript
- Vite 7
- Redux Toolkit
- Material-UI v7
- Axios
- PWA (Progressive Web App)

### Backend
- Spring Boot
- JPA/Hibernate
- H2 Database
